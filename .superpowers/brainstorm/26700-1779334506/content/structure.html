Research Page — Template Structure

No text, no images — just structural containers. Which layout?

A

Research Index (Like Programs)

Header + tabbed interface. Each tab = one research area. Tab panel has: title placeholder, description block, two-column grid, terminal readout. Matches programs page structure.

B

Research Registry (Terminal-First)

Header + vertical list of research entries. Each entry: label, title placeholder, description block, status tag. Terminal sidebar on right. Like a database listing.

C

Featured + Archive Split

Header + featured research spread (two-column hero) + archive grid below. Featured section has large placeholder area. Archive section has smaller entry cards.
