Key Features

  • CSS-only tabs using :target — no JS
  • Each program gets full detail: curriculum, requirements, faculty, research
  • Terminal readout per program for worldbuilding flavor
  • Dark page throughout — no parchment section

Trade-offs

  • :target means URL changes on tab switch (back button works)
  • Default tab requires a redirect or hash in URL
  • Longer page per program — more content to write
