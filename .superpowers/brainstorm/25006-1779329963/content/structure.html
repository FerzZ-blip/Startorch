Hybrid Approach — Structure

Index landing + expandable deep-dive sections. Which structure feels right?

A

Anchor-Scroll

Top nav with program links (Synchronists, Light Energy, Exostrider). Clicking scrolls to that program's full section. Each section has: quote, description, curriculum, requirements, faculty, terminal readout. Single page, no JS tabs.

B

Expand/Collapse

Programs listed as entries (like index.html). Clicking expands inline to reveal curriculum, requirements, terminal data. Only one open at a time. More compact, feels like a terminal registry.

C

Tabbed Sections

Tab bar at top (pure CSS :target hack). Each tab shows one program's full detail. Clean, focused, one program at a time. No JS needed.
